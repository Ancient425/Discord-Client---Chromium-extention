This is an chromium extension 

Purpose :- 
    to edit the looks of your discord client
    and it looks good imo

    Changes that it makes : - 
        1. changes the background of the chat with a blured image
        2. changes the look of the members list with a gradient
        3. gives a gradient to the title bar of the chat
        4. redoes color pallet of discord
        5. pixel font

```
How to use :-
    0. download and extract the repo in a folder 
    1. go to [chrome://extensions] of any chromium based browser , eg - brave , edge , chrome , helium .
    2. enable "developer mode" .
    3. click on load unpacked
    4. selected the folder in which you have extracted the zip file of this repo
    5. boom , done , (refresh the extension js in case)
```
